# Solace — flow prototype

A self-contained, animated three-screen walkthrough of the Solace companion:

1. **Morning reflection** — daily verse + companion greeting
2. **Prep for the visit** — uploads a visit summary, drafts questions for the doctor
3. **In the room** — live recording, transcript, and the prepared questions checking off

Navigate with the arrows / dots under the phone, the arrow keys, or by tapping a starter chip on screen 1. Single `index.html`, no build step.
